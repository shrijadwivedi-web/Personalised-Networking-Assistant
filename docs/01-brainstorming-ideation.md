# 1. Brainstorming & Ideation

## Problem Statement

Networking events are often crowded, fast-moving, and stressful. People may know the event topic
but still struggle to start a useful conversation that feels relevant and natural.

## Idea

Build an AI-powered assistant that turns an event description and a user's interests into tailored
conversation starters, fact-check summaries, and follow-up context.

## Why This Project

- Helps users feel more prepared before professional or social networking events
- Reduces awkward first interactions
- Makes conversation suggestions more personal by using role, industry, and goals

## Outcome of Ideation

The final concept became a full web app with:

- login and registration
- profile-based personalization
- topic extraction from event descriptions
- conversation starter generation
- quick fact checking
- history and feedback tracking
