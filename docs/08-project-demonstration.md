# 8. Project Demonstration

## Demo Flow

1. Show the login and registration flow
2. Open the profile page and fill in user details
3. Generate starters from an event description
4. Review confidence scores and explanations
5. Try the fact-check page
6. Open history and feedback pages

## Talking Points

- The app is personalized, not generic
- The system combines local analysis with hosted generation
- User data is scoped to the account that created it
- The interface is designed for repeated use at real events

## Future Improvements

- Exportable reports
- Richer profile customization
- Better analytics on suggestion quality
- Additional source-backed fact-checking options
