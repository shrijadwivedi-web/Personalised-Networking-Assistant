# Define Problem Statements

Networking events often leave people with only a few seconds to make a useful first impression.
Users need quick, context-aware conversation starters that feel relevant to the event and to their
own background.

## Problem

- Users struggle to start conversations at professional or social events
- Generic icebreakers do not reflect the event topic or the user's interests
- People want a fast way to fact-check topics before speaking

## Goal

Build an assistant that turns an event description into personalized, useful, and confident
conversation starters.
